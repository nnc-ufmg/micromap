Interface Functions module
==========================

This module contains functions that are used to interface with the rhd2000 chip.

All functions listed here are linked to the electrophysiological recording system interface that uses the RHD2000 chip
This part contains the backend of the package. Herein lies the the parameters programmed by the user 
that will be processed, allocated and modified within the interface of the package.
